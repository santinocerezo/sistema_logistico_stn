## 💡 NOTAS IMPORTANTES

### Tailwind v3 vs v4
- **v4:** Nueva sintaxis, aún inestable, causaba errores
- **v3:** Sintaxis tradicional, estable, ampliamente soportada
- **Decisión:** Usar v3.4.1 para máxima compatibilidad

### Caché del Navegador
Si ves página en blanco:
1. Presiona `Ctrl+Shift+R` (Windows/Linux)
2. O `Cmd+Shift+R` (Mac)
3. Esto hace un hard refresh y limpia el caché

### Hot Module Replacement (HMR)
- Vite tiene HMR activo
- Los cambios se reflejan automáticamente
- No necesitas reiniciar el servidor

## 🔍 TROUBLESHOOTING

### Si la página sigue en blanco:
1. Abre DevTools (F12)
2. Ve a la pestaña Console
3. Busca errores en rojo
4. Comparte el error conmigo

### Si hay errores de CSS:
1. Verifica que Tailwind v3.4.1 esté instalado
2. Ejecuta: `npm list tailwindcss`
3. Debería mostrar: `tailwindcss@3.4.1`

### Si el servidor no inicia:
1. Detén el proceso: Ctrl+C
2. Limpia node_modules: `rm -rf node_modules`
3. Reinstala: `npm install`
4. Inicia: `npm run dev`
